Tangensfunktionen

📚 Lärandemål

  • Förstå definitionen av tangens via enhetscirkeln
  • Skissa grafen för $y = \tan x$ och identifiera dess egenskaper
  • Lösa tangentekvationer

Definition och Egenskaper

Definition
$$\tan v = \frac{\sin v}{\cos v}$$
EgenskapVärde
Period$180°$ (eller $\pi$)
DefinitionsmängdAlla $x \neq 90° + n \cdot 180°$
VärdemängdAlla reella tal
Asymptoter$x = 90°, 270°, ...$
⚠️ Viktigt

Tangensfunktionen är odefinierad när $\cos x = 0$, dvs vid $x = 90° + n \cdot 180°$.

Interaktiv Graf

Observera tangensgrafens asymptotiska beteende och periodicitet:

$y = \tan(x)$ — Notera de vertikala asymptoterna vid $90°$, $270°$ etc.

Lösa Tangentekvationer

Allmän lösning: $\tan v = a$
$$v = \arctan(a) + n \cdot 180°$$
Exempel
Lös $\tan x = 1$ för $x \in [0°, 360°]$
Lösning
$$\tan x = 1$$ $$x = \arctan(1) = 45°$$ $$\text{Alla lösningar: } x = 45° + n \cdot 180°$$ $$\text{I } [0°, 360°]: x = 45° \text{ eller } x = 225°$$

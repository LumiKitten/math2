Lektion 16

Sinus- och Cosinusfunktioner

📚 Lärandemål

  • Skissa graferna för $y = \sin x$ och $y = \cos x$
  • Förstå begreppen amplitud, period och medelvärde
  • Analysera funktioner av typen $y = A\sin(Bx) + C$

Grundläggande Egenskaper

Egenskap$y = \sin x$$y = \cos x$
Period$360°$ (eller $2\pi$)$360°$ (eller $2\pi$)
Amplitud$1$$1$
Medelvärde$0$$0$
Maxvärde$1$$1$
Minvärde$-1$$-1$

Interaktiv Graf

Experimentera med reglagen för att se hur parametrarna A, B och C påverkar grafen:

$y = A \cdot \sin(B \cdot x) + C$ — Dra i reglagen för A, B, C

Transformationer

Generell form
$$y = A\sin(Bx + \varphi) + C$$
ParameterPåverkan
$A$Amplitud = $|A|$
$B$Period = $\frac{360°}{|B|}$ (eller $\frac{2\pi}{|B|}$)
$C$Vertikal förskjutning (medelvärde)
$\varphi$Fasförskjutning (horisontell)

Genomgångna Exempel

Exempel
Bestäm amplitud, period och medelvärde för $y = 3\sin(2x) + 1$.
Lösning
$$A = 3 \Rightarrow \text{Amplitud} = 3$$ $$B = 2 \Rightarrow \text{Period} = \frac{360°}{2} = 180°$$ $$C = 1 \Rightarrow \text{Medelvärde} = 1$$ $$\text{Maxvärde} = 1 + 3 = 4, \quad \text{Minvärde} = 1 - 3 = -2$$

Lektion 19

Kurvan $y = a\sin x + b\cos x$

Omskrivningsformel

Huvudformel
$$a\sin x + b\cos x = R\sin(x + \varphi)$$ $$\text{där } R = \sqrt{a^2 + b^2} \text{ och } \tan \varphi = \frac{b}{a}$$
Exempel
Skriv $3\sin x + 4\cos x$ på formen $R\sin(x + \varphi)$.
Lösning
$$R = \sqrt{9 + 16} = 5$$ $$\tan \varphi = \frac{4}{3} \Rightarrow \varphi \approx 53{,}1°$$ $$\text{Svar: } 5\sin(x + 53{,}1°)$$

📝 Övningar

  1. Skriv $\sin x + \cos x$ på formen $R\sin(x + \varphi)$.
  2. Bestäm max- och minvärde för $y = 3\sin x - 4\cos x$.
1. $\sqrt{2}\sin(x + 45°)$
2. Max = 5, Min = -5

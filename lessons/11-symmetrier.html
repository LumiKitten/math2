Exempel
Beräkna $\sin 225°$ exakt.
Lösning
$$225° = 180° + 45°$$ $$\sin 225° = -\sin 45° = -\frac{\sqrt{2}}{2}$$

📝 Övningar

  1. Beräkna exakt: $\cos 315°$
  2. Beräkna exakt: $\sin 210°$
  3. Förenkla: $\sin(90° - v) + \cos(90° - v)$
  4. Om $\sin v = \frac{3}{5}$ och $90° < v < 180°$, bestäm $\cos v$.
1. $\cos 315° = \cos(-45°) = \frac{\sqrt{2}}{2}$
2. $\sin 210° = -\sin 30° = -\frac{1}{2}$
3. $\cos v + \sin v$
4. $\cos v = -\frac{4}{5}$ (negativ i kvadrant II)
